Start with cloning this repo on your local machine

To install and set up the library, run:

```sh
$ node --version
$ npm install
```
### ⚠️ run "npm install" in the frontend and backend folder also. ⚠️

## **Usage**

### Serving the app

```sh
$ npm start ( backend)
$ npm run dev (frontend)
```
### Running the tests

```sh
$ npm test
```
